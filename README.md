## Commits

*Install the following utility:*

Conventional commit utility: [git-cz](https://github.com/commitizen/cz-cli)

```
npm install -g commitizen
```

*To commit new changes use*
```
git cz
```

## Tooling management

Volta:

```bash
# install Volta
curl https://get.volta.sh | bash

# install node
volta install node

# start using Node
node
```
